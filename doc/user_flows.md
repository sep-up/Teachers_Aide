# Teachers Aide - 用户流程图

## 1. 用户登录流程

```mermaid
flowchart TD
    A[教师打开小程序] --> B[首次进入?]
    B -->|是| C[微信授权登录]
    B -->|否| D[自动登录]
    C --> E[调用wx.login获取code]
    E --> F[云函数换取openid]
    F --> G[查询用户是否存在]
    G -->|不存在| H[创建新用户]
    G -->|存在| I[获取用户信息]
    H --> J[存入全局状态]
    I --> J
    D --> J
    J --> K[进入首页]
```
