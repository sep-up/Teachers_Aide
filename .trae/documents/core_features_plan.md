# Teachers Aide - 核心功能精简计划

## 目标
控制云函数的量，只保留核心功能：课程表和通知功能，删除考勤记录、成绩录入、学生管理相关的文件和代码。

## 任务列表

### [x] 任务1：删除不需要的云函数
- **优先级**：P0
- **描述**：删除考勤记录、成绩录入、学生管理相关的云函数
- **需要删除的文件**：
  - `miniprogram/cloud/student/` 目录
  - `miniprogram/cloud/class/` 目录
- **成功标准**：只保留 `login`、`openapi`、`notice`、`schedule` 云函数

### [x] 任务2：删除不需要的页面
- **优先级**：P0
- **描述**：删除考勤记录、成绩录入、学生管理相关的页面
- **需要删除的文件**：
  - `miniprogram/pages/attendance/` 目录
  - `miniprogram/pages/scores/` 目录
  - `miniprogram/pages/students/` 目录
  - `miniprogram/pages/class/` 目录
  - `miniprogram/pages/comments/` 目录
- **成功标准**：只保留 `index`、`schedule`、`notice`、`profile` 页面

### [x] 任务3：修改 app.json 配置
- **优先级**：P0
- **描述**：更新 app.json 文件，移除不需要的页面配置
- **修改内容**：
  - 从 pages 数组中移除不需要的页面路径
  - 保持 tabBar 配置只包含首页、课程表、通知、我的
- **成功标准**：app.json 中只包含保留的页面配置

### [x] 任务4：修改 index 页面
- **优先级**：P1
- **描述**：更新 index 页面，移除跳转到已删除功能的入口
- **修改内容**：
  - 移除跳转到班级管理、学生管理、成绩录入、评语生成、考勤记录的按钮
  - 只保留跳转到课程表和通知的入口
- **成功标准**：index 页面只包含核心功能的入口

### [/] 任务5：修改 profile 页面（如果需要）
- **优先级**：P2
- **描述**：更新 profile 页面，移除与已删除功能相关的内容
- **成功标准**：profile 页面只包含与核心功能相关的内容

### [ ] 任务6：测试核心功能
- **优先级**：P0
- **描述**：测试课程表和通知功能是否正常运行
- **测试内容**：
  - 课程表的添加和显示
  - 通知的发布和显示
  - 通知的转发功能
- **成功标准**：核心功能正常运行，无错误

## 执行步骤

4. 执行任务4：修改 index 页面

- 项目只包含课程表和通知两个核心功能
- 云函数数量减少，只保留必要的函数
